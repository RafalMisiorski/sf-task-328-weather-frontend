# weather-react

Auto-generated frontend application by SW Frontend Generator.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
```

### Type Check

```bash
npm run type-check
```
